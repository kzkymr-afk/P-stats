# マスタ関連の議論まとめ（Gemini打ち合わせ用）

このドキュメントは、マスターデータの扱いについて行った議論を整理したものです。別途 Gemini と打ち合わせしながら進める際の前提として使ってください。

---

## 1. 方針（合意したこと）

- **マスタの正本はアプリの外（Web上）に置く**
  - イメージ: アプリの外部の Web 上にマスターデータを置き、アプリがそれを参照しに行く形にする。
- **管理人（あなた）だけがマスタを編集する**
  - アプリ内に「マスタ管理」画面は出さない（一般ユーザーには見せない）。
- **無料で運用したい**
  - まずは GitHub の Raw URL で JSON を配る方法を案内済み。

---

## 2. すでにやったこと（実装・変更）

| 内容 | 詳細 |
|------|------|
| **ホームから「マスタ管理」を非表示** | `P_statsApp.swift` の HomeView から「マスタ管理」の NavigationLink を削除済み。アプリのメニューには出ていない。 |
| **アプリ内マスタ編集は残している** | `PresetMasterListView` / `PresetMachineEditView`（PresetMasterView.swift）はコード上は残しているが、どこからも遷移しない。必要なら後から「隠し起動」や DEBUG 時のみ表示などに流用可能。 |
| **サーバー取得の仕組みは既存のまま** | `PresetServiceConfig.presetListURL` に URL を入れると、`PresetService.fetchPresets(from:)` で GET して JSON をパース。`MasterMachineSearchView` が「マスタから検索」でその一覧を表示し、採用でマイリストに追加する流れは実装済み。 |

---

## 3. 現在のコード構成（マスタまわり）

| ファイル | 役割 |
|----------|------|
| **PresetServiceConfig.swift** | マスタ取得元 URL。`presetListURL` が空ならアプリ内プリセット（PresetMachine）を参照。ここに Web の URL を設定するとサーバー取得になる。 |
| **PresetService.swift** | `fetchPresets(from: String)` で URL に GET し、`[PresetFromServer]` にデコード。`PresetFromServer` の定義もここ。 |
| **MasterMachineSearchView.swift** | マイリスト画面から「マスタから検索して追加」で開く画面。`presetListURL` が空でなければサーバーから取得して一覧表示。 |
| **PresetMasterView.swift** | アプリ内マスタ編集（PresetMachine の CRUD）。現在はどこからもリンクしていない。 |
| **MachineEditView.swift** | 機種編集画面。サーバープリセット一覧の参照に `PresetServiceConfig.presetListURL` を使っている箇所あり。 |

## 5. 今後の打ち合わせで決めるとよいこと（例）

- **マスタの置き場所**
  - GitHub Raw のままにするか、Vercel / Netlify / その他に移すか。
- **URL の設定方法**
  - コードに直書きのままか、Info.plist や環境変数で切り替えるか。
- **アプリ内マスタ編集（PresetMasterView）**
  - 完全に使わず削除するか、DEBUG 時のみや隠し操作で管理者だけ使えるようにするか。
- **JSON のスキーマ**
  - `PresetFromServer` の項目で足りないもの・変えたいものがないか。
- **エラーハンドリング・オフライン**
  - 取得失敗時やオフライン時の表示・キャッシュの要否。

---

## 6. 用語の対応

- **マスタ** = 機種プリセットの「正本」。Web 上に 1 本置く想定。
- **presetListURL** = その正本を取得する URL（`PresetServiceConfig.presetListURL`）。
- **PresetFromServer** = サーバー（Web）から返す JSON 1 件分の型。
- **PresetMachine** = アプリ内 SwiftData のマスタ機種（サーバーを使う場合は補助 or 未使用）。
- **マスタから検索** = ユーザーがマイリストに機種を追加するときに、マスタ（Web またはアプリ内）の一覧から選ぶ機能。画面は `MasterMachineSearchView`。

このまとめを Gemini に渡せば、マスタ関連の前提と現状を共有した状態で打ち合わせを続けられます。
