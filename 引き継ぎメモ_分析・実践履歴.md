# 引き継ぎメモ（分析・実践履歴まわり）

新しいチャットで続きの作業をするときのための要点まとめです。

---

## プロジェクト概要

- **P-stats**: パチンコ実戦記録・ボーダー分析アプリ（SwiftUI + SwiftData）
- 主な画面: 遊戯画面（PlayView）、機種・店舗選択（MachineShopSelectionView）、分析（AnalyticsDashboardView）、実戦履歴（P_statsApp 内 HistorySessionCard など）

---

## ここまでに実装した主な機能

### 1. 日別分析の削除 → 特定日傾向分析へ

- **削除**: 分析の「日別」セグメント、`AnalyticsEngine.byDay` / `dayKey`
- **追加**: 店舗ごとの「特定日」ルールに基づく集計
  - **Date 共通関数**（`AnalyticsEngine.swift` の `SpecificDateUtils`）:
    - `dayLastDigit(from:)` … 日の下一桁
    - `isZoroMe(from:)` … ゾロ目（月＝日）
    - `attributeLabel(for:rules:)` … 「通常日」「毎月N日」「Nのつく日」「ゾロ目」のラベル返却
  - **Shop**（`Models.swift`）: `specificDayOfMonthStorage`, `specificLastDigitsStorage`（カンマ区切りで最大2つずつ想定）
  - **店舗編集**（`MachineShopSelectionView` の `ShopEditView`）: 「毎月この日 ①/②」「Nのつく日 ①/②」の4入力
  - **分析**: 特定日傾向の BarChart（期待値・回転率を属性別に比較）。未入力のルール区分は表示しない。ラベルは「毎月3日」「7のつく日」のように数値代入済みで表示

### 2. 実践履歴パネルの表示変更

- **文言**: 「実践ボーダー」→「実践回転率」（回/千円）、「理論」→「理論期待値」
- **欠損・余剰**: プラスは「余剰 +N円」、マイナスは「欠損 N円」のみ表示
- **大当たり**: 「大当たり RUSH:n 通常:m」を表示（`GameSession.rushWinCount` / `normalWinCount`）
- **レイアウト**: 実践回転率・理論期待値・余剰 or 欠損を `miniblock` で縦並びにし、折り返し防止
- **GameSession**（`Models.swift`）: `rushWinCount`, `normalWinCount`, `formulaBorderPer1k` を追加。保存時（`PlayView.saveCurrentSession`）で `log.rushWinCount` / `normalWinCount` と公式ボーダー（`parseFormulaBorder`）を渡している
- **実践回転率の括弧**: 公式ボーダーがあるとき「250.5 回/千円 (+12.3)」のように差を表示

### 3. その他の修正

- **機種編集**（`MachineEditView`）: スクロールが上に戻る問題 → `GeometryReader` を Web 表示部分だけに限定
- **通常モード・遊戯終了ボタン**: 下にはみ出る赤を消すため、該当 `ZStack` に `.clipped()`

### 4. 分析ページ上部の通算サマリ

- **表示順**: ① 通算実収支 → ② 通算実践回転率（加重平均）→ ③ 公式ボーダーとの差の平均
- **通算実収支**: 現在のフィルタ対象セッションの `profit` 合計
- **通算実践回転率**: 加重平均 = 総回転数 ÷ (総実質投資/1000)
- **公式ボーダーとの差の平均**: 各セッションの (実践回転率 − formulaBorderPer1k) の単純平均（公式未設定は除外、該当なしは「—」）
- フィルタ（店舗・機種・期間・メーカー）ありのときは、そのグループだけを対象にした通算

## 用語・データの意味

- **実践回転率**: 回転数 ÷ (実質投資/1000) ＝ 回/千円
- **公式ボーダー**: 機種の等価ボーダー（回/千円）。`GameSession.formulaBorderPer1k` に保存時点の値を保存
- **特定日ルール**: 店舗ごと。毎月この日（1〜31）、Nのつく日（0〜9）。未入力の区分は分析に出さない
- **通算の「加重平均」**: 実践回転率は「総回転数 ÷ (総実質投資/1000)」で算出（投資額で加重した形）
